## Overview

This repository contains the **Week 6 Thursday assignment** completed as part of the Sohail Smart Solutions Summer Training Program 2026.

The objective of this assignment is to develop a cloud data protection plan for the fictional organization **EduTrack Learning Solutions** as it continues migrating its Learning Management System (LMS) to **Amazon Web Services (AWS)**. The report focuses on protecting sensitive cloud data through encryption, key management, data classification, privacy controls, and compliance with international standards and regulations.

This assignment continues the same AWS migration scenario developed throughout the previous Week 6 tasks.

## Learning Objectives

This assignment focuses on the following cloud security concepts:

- Understand cloud data classification.
- Apply encryption at rest and encryption in transit.
- Learn the purpose of AWS Key Management Service (KMS).
- Understand tokenization and key management.
- Explain data residency and data sovereignty.
- Study GDPR and UAE Personal Data Protection Law (PDPL).
- Understand ISO/IEC 27018 for protecting Personally Identifiable Information (PII).
