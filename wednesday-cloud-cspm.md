## Overview

This repository contains the **Week 6 Wednesday assignment** completed as part of the Sohail Smart Solutions Summer Training Program 2026.

The objective of this assignment is to understand how secure cloud configuration baselines and **Cloud Security Posture Management (CSPM)** help organizations continuously detect and remediate cloud misconfigurations during an AWS migration.

This task continues the cloud migration scenario for **EduTrack Learning Solutions**, building on the previous assignments covering cloud security foundations and Identity and Access Management (IAM).
