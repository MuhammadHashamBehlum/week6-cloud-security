## Overview

This repository contains the Week 6 Monday assignment completed as part of the Sohail Smart Solutions Summer Training Program 2026.

The objective of this assignment is to understand cloud security fundamentals by exploring the Shared Responsibility Model, identifying cloud-specific threats, and developing a cloud threat model for a fictional organization migrating its services to Amazon Web Services (AWS).

The migration scenario continues the Governance, Risk, and Compliance (GRC) work completed in previous internship tasks using the fictional organization **EduTrack Learning Solutions**.

---

## Learning Objectives

- Understand the Shared Responsibility Model.
- Differentiate between cloud provider and customer responsibilities.
- Learn common cloud security threats.
- Understand cloud governance principles.
- Explore ISO/IEC 27017 and ISO/IEC 27018.
- Build a basic cloud threat model.
